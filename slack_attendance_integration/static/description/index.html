THE PROBLEM

Attendance tracking shouldn't be a second job

Switching apps, forgetting to log in, manual HR entries — it adds up to wasted time and inaccurate data.

Before

Open browser → find Odoo → click Attendance → click Check In → go back to Slack

After

Type /login in Slack. Done. Odoo attendance updated instantly.

SLASH COMMANDS

Four commands. That's all your team needs.

/login

Records attendance in Odoo. Optional grace minutes applied automatically.

/logout

Closes the attendance record. Net working hours calculated instantly.

/break

Starts a timed break. Short breaks under the threshold are not deducted.

/resume

Returns from break. Break duration calculated and saved to Odoo.

SMART FEATURES

Thoughtful details that matter.

Login Grace Minutes

Type /login at 10:00 AM with a 5-minute grace window? Odoo records 9:55 AM. No more penalising employees for being 30 seconds late.

Minimum Break Threshold

Breaks under the configured minimum (e.g. 5 min) are saved but not deducted from working hours.

Net Working Hours

Net = Total − Counted Breaks — shown live in the Odoo Attendance list and form view.

HMAC-SHA256 Verification

Every Slack request verified with Signing Secret. Replay attacks blocked with 5-minute timestamp window.

Per-Employee Timezone

Times shown in each employee's own timezone — read from their HR record. No hardcoded IST.

Missed Logout Alerts

Forgot to /logout? Get an automatic Slack DM reminder at end of day. No more open attendance records.

AI FEATURE

Personalised daily summaries via Claude AI

Enable AI mode with an Anthropic API key. Each employee gets a tailored Slack DM — referencing their actual hours, streak, and target.

  • Streak tracking — consecutive days on target
  • Plain-text fallback if AI is disabled
  • Summary cached on attendance record
  • API key visible to Odoo admins only

Slack Attendance Integration

Let employees check-in, take breaks, and check-out using simple Slack slash commands — directly synced to Odoo HR Attendance.

/login /break /resume /logout

How It Works

Employees use 4 Slack slash commands. Every action is instantly recorded in Odoo — no forms, no manual entries.

Step 01
/login
Records check-in. Optional grace minutes can shift the recorded time (e.g. /login at 9:05 → recorded as 9:00).
Step 02
/break
Starts a break session. Employee must be logged in. Blocked if a break is already active.
Step 03
/resume
Ends the active break. Breaks shorter than the configured minimum are saved but not deducted from hours.
Step 04
/logout
Records check-out time. Cannot proceed if employee is still on an active break.

Features

Smart rules, clean data, and real-time Slack feedback for every attendance action.

⚡
4 Slack Slash Commands
Complete attendance lifecycle — login, break, resume, logout — handled entirely from Slack without switching apps.
🕐
Login Grace Minutes
Configure a grace period. If set to 5 minutes, a /login at 9:05 AM records as 9:00 AM — fair to employees, accurate for HR.
☕
Smart Break Tracking
Set a minimum break duration. Breaks shorter than the threshold are logged but not deducted from working hours.
📊
Net Working Hours
Auto-calculated on each attendance record: total hours minus counted break time = net worked hours.
📬
Daily Summary DM
At end of day, each employee gets a personalized Slack DM with check-in/out times, break duration, and a flag if hours are below the minimum.
🔐
Secure Webhook Handling
Slack Signing Secret verification on every request. SSL configured for Odoo.sh compatibility. No request is processed without validation.

Configuration

One configuration record inside Odoo. No external tools required.

Slack Credentials
Signing SecretVerifies every incoming Slack request
Bot Tokenxoxb- token used to send DMs to employees
Webhook URLIncoming webhook for channel notifications
Attendance Rules
Grace MinutesShift login time back by N minutes (default: 5)
Min Break MinutesMinimum break to deduct from hours (default: 5)
Min Working HoursExpected hours/day for daily summary flag (default: 8h)
Employee Setup
Slack User IDAdd each employee's Slack User ID (e.g. U01234ABCDE) on their HR Employee profile to link their Slack account.
Slack App Setup (Required)
EndpointPOST /slack/attendance
Register this URL as the Request URL for each slash command in your Slack App's configuration panel. This module requires a Slack App with slash commands and a bot token.

Requirements & Dependencies

This module requires an active Slack workspace with a configured Slack App (slash commands + bot token + incoming webhook). No paid external services are required beyond your Slack plan.

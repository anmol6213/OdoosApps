Configuration

  • Add Slack Bot Token & Signing Secret in Odoo Settings
  • Map employee Slack User IDs
  • Set grace time & timezone preferences
  • Enable/disable AI summary feature

Security

All requests are verified using Slack's HMAC-SHA256 signature — no unauthorized access possible.

🔗 Slack Attendance Integration

Track employee attendance directly from Slack using simple slash commands. Fully synced with Odoo HR Attendance.

Odoo 19.0 Free LGPL-3 HR / Attendances
/login /logout /break /resume

Overview

This module connects your Slack workspace with Odoo HR Attendance. Employees can check in, check out, and manage breaks using slash commands — no need to open Odoo. All records are saved automatically and visible in Odoo's attendance reports.

Features

  • ⚡
    4 Slack Slash Commands

    Employees use /login, /logout, /break, and /resume directly from Slack. Records appear instantly in Odoo.

  • 🕐
    Login Grace Minutes

    Set a grace window (e.g. 5 min) so /login at 9:00 AM records as 8:55 AM. Removes clock-racing pressure.

  • ☕
    Minimum Break Threshold

    Short breaks under the configured limit (e.g. 5 min) are saved but NOT deducted from working hours.

  • 📊
    Net Working Hours

    Each attendance record shows: Net Hours = Total Hours − Counted Breaks. Visible in Odoo attendance view.

  • 🔔
    Daily Summary DM

    Employees get an end-of-day Slack message with check-in time, check-out time, and total hours worked.

  • ⚠️
    Missed Logout Alerts

    Employees who forgot to /logout receive an automatic Slack reminder. Keeps attendance data clean.

  • 🌍
    Per-Employee Timezone

    All times shown in each employee's own timezone. Works for global teams — no hardcoded region.

  • 🤖
    AI Daily Summaries (Optional)

    Enable personalized AI messages via Google Gemini (free) or Anthropic Claude (paid). Mentions hours, streak, and motivation.

  • 🔐
    Secure — HMAC-SHA256 Verification

    Every Slack request is verified before any action. Replay attacks blocked. SSL uses certifi — no CERT_NONE.

How to Use

After installation, employees simply type these commands in Slack:

  • 🟢
    /login

    Check in. Records attendance in Odoo with optional grace time applied.

  • 🔴
    /logout

    Check out. Closes the attendance record. Cannot logout while on break.

  • 🟡
    /break

    Start a break. Must be logged in first.

  • 🔵
    /resume

    End the break and return to work. Short breaks under threshold are not counted.

Installation & Setup

  1. Install the module

    Copy slack_attendance_integration to your Odoo addons path. Restart server. Install from Apps menu.

  2. Configure Slack credentials

    Go to Attendances → Configuration → Slack Settings. Enter Signing Secret, Bot Token, and Webhook URL.

  3. Set webhook URL in Slack App

    In Slack App settings, point all 4 slash commands to: https://your-odoo.com/slack/attendance

  4. Link employees to Slack

    Open each employee's HR profile → add their Slack User ID (Slack → profile → Copy member ID).

  5. Done — start using commands

    Employees can now use /login, /logout, /break, /resume from any Slack channel.

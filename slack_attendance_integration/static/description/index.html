THE PROBLEM

Attendance tracking shouldn't be a second job

Switching apps, forgetting to log in, manual HR entries — wasted time and inaccurate data.

Before

Open browser → find Odoo → click Attendance → Check In → back to Slack

After

Type /login in Slack. Done. Odoo updated instantly.

SLASH COMMANDS

Four commands. That's all your team needs.

/login

Records attendance. Grace minutes applied automatically.

/logout

Closes record. Net working hours calculated instantly.

/break

Starts a timed break. Short breaks not deducted.

/resume

Returns from break. Duration saved to Odoo.

SMART FEATURES

Thoughtful details that matter.

Login Grace Minutes

Type /login at 10:00 AM with 5-min grace? Odoo records 9:55 AM.

Minimum Break Threshold

Breaks under minimum saved but not deducted from working hours.

Net Working Hours

Net = Total − Counted Breaks shown live in Odoo Attendance views.

HMAC-SHA256 Verification

Every request verified with Signing Secret. Replay attacks blocked.

Per-Employee Timezone

Times in employee's own timezone from HR record. No hardcoded IST.

Missed Logout Alerts

Auto Slack DM reminder if forgot /logout. No dangling records.

AI FEATURE

Personalised daily summaries via Claude AI

Each employee gets a tailored Slack DM referencing their actual hours, streak, and target.

  • Streak tracking — consecutive days on target
  • Plain-text fallback if AI is disabled
  • Summary cached on attendance record
  • API key visible to Odoo admins only

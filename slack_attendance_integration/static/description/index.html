Overview

Slack Attendance Integration allows employees to mark attendance directly from Slack using simple commands: /login, /break, /resume, /logout. It integrates seamlessly with Odoo's HR Attendance module with secure HMAC verification, timezone support, grace time, and optional AI summary.

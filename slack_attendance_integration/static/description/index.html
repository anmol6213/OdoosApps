Overview

This module allows employees to manage attendance directly from Slack using simple commands. No need to open Odoo — everything is handled inside Slack.

Features

  • /login Check-in with grace time
  • /break Start break
  • /resume End break
  • /logout Check-out with net hours
  • Timezone support per employee
  • Break threshold configuration
  • Missed logout alerts
  • Optional AI daily summary
  • Secure Slack request verification (HMAC-SHA256)

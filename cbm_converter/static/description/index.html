Overview

CBM Converter automatically calculates the number of pieces from CBM (Cubic Meter) quantity for construction materials like AAC Blocks. It extracts dimensions directly from the product name (e.g. 600x200x225) and computes the correct piece count — no manual entry needed.

Works seamlessly across Sale Orders, Customer Invoices, and Stock Deliveries. Bidirectional: enter CBM to get Pieces, or enter Pieces to get CBM.

Key Features

📐

Auto Dimension Parsing

Extracts L×W×H from product name automatically. No extra configuration needed.

🔄

Bidirectional Conversion

Enter CBM → get Pieces. Enter Pieces → get CBM. Both directions work seamlessly.

⚡

No Infinite Loop

Smart onchange logic prevents quantity from overwriting itself — no more looping issues.

How It Works

1

Product Name Contains Dimensions

Product name must have dimensions in format like 600x200x225 (L×W×H in mm). Example: "INFRA.MARKET AAC BLOCKS 600x200x225 Grade 1"

2

Enter Quantity (CBM)

Enter the quantity in CBM on Sale Order, Invoice, or Delivery. Pieces field auto-calculates instantly.

3

Or Enter Pieces

Enter pieces directly — CBM quantity updates automatically. Perfect for ordering by piece count.

Calculation Example

# Product: AAC BLOCKS 600x200x225 # Volume per block = 600mm × 200mm × 225mm vol = (600 × 200 × 225) / 1_000_000_000 = 0.027 CBM per piece # CBM → Pieces qty = 7.67 CBM pieces = ceil(7.67 / 0.027) = ceil(284.07) = 285 pcs ✅ # Pieces → CBM pieces = 285 qty = 285 × 0.027 = 7.695 CBM ✅
